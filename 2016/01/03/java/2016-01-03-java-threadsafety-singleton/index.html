进程安全的Java单例

  • java
java

进程安全的单例是singleton类的java对象都返回一样的实例，尤其在多进程环境下。比较典型的例子是java.lang.Runtime，这就是一个单例的类，用的是静态域的方式。

枚举

用枚举来创建单例是目前最简单的方式。在java中，枚举实例默认是final的。

静态域

可以声明一个静态域，在类加载的时候就将类实例化到内存中，但是这样有一点不好，就是这不是懒加载，不管系统中是否会用到这个对象，都会被实例化。
